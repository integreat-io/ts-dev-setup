# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 6.0.x   | :white_check_mark: |
| < 6.0   | :x:                |

## Reporting a Vulnerability

We welcome vulnerabilty reports. Please
[open an issue](https://github.com/integreat-io/ts-dev-setup/issues/new) and
describe what you've find.
